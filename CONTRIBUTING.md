# Contributing

Thanks for helping improve the Session Helper APIHelper bridge.

## Development Guidelines

- Keep this package focused on adapting Session Helper to APIHelper.
- Do not change APIHelper from this package.
- Do not change Session Helper core runtime APIs from this package.
- Do not add backend-specific login, refresh, or API request behavior.
- Add XML documentation for public runtime APIs.
- Add editor tests for behavior changes.

## Testing

Run the package editor tests from Unity's Test Runner in a project that has Session Helper, APIHelper, and this bridge package installed.
